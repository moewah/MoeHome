# Changelog

All notable changes to this project will be documented in this file.

## [v2.1.0] - 2026-03-21

### 重构

- **音乐播放器** - 重构主页播放器组件，与动态页面音频播放器 UI 风格统一
- **移动端导航** - 重构移动端菜单导航，统一下拉菜单样式及菜单项字体图标规范

### 优化

- 冗余代码清理

## [v2.0.0] - 2026-03-20

### 新功能

- **Memos 动态模块** - 全新 Moments 页面，集成 Memos API
  - 时间线布局 + 玻璃卡片设计
  - 标签筛选（自动收集 + 点击切换）
  - 图片灯箱（支持键盘导航、触摸滑动）
  - 音频播放器（自定义控件 + 波形装饰）
  - 视频播放器（自定义控件 + 全屏支持 + iOS 兼容）
  - 文档附件预览（PDF/Word/Excel 等）
  - 完整 Markdown 渲染（代码高亮 + 表格 + TODO 列表）
  - 无限滚动（IntersectionObserver 实现）
  - 置顶动态支持
- **主题数据模块** - 提取 `theme-data.js`，配色方案构建/运行时共享
- **模板 Partials** - 新增 `navbar.html`、`footer.html` 模板片段
- **留言页面** - 新增独立 Guestbook 页面，终端风格弹幕交互

### 重构

- **主题系统** - 配色方案集中管理，衍生色自动计算
- **模板架构** - 引入 partials 机制，消除重复代码

### 优化

- **留言板** - 修复终端回复交互，优化弹幕项目可读性
- **贡献图** - 新增禁用状态处理、useRealData 配置项
- **构建脚本** - 贡献配置提取与条件渲染

### 其他

- 删除过时的截图和计划文档
- 新增支付宝收款码图片
- README 文档同步更新

## [v1.2.0] - 2026-03-14

### 新功能

- **终端装备列表** - 新增 `gear` 配置项，支持 `cat gear.txt` 命令展示个人装备
- **全局效果封装** - 将光标、背景视差、回到顶部、元素光晕封装为独立类（CustomCursor、GridParallax、ScrollProgress、ElementGlow），自动创建元素，多页面复用
- **404 页面优化** - 容器结构与首页统一，文案优化

### 优化

- **导航链接** - 统一使用 `/` 路径，适配部署环境
- **RSS 获取** - 超时时间从 10s 增加到 20s，提高稳定性
- **代码结构** - 移除 config.js 中 scrollProgress 配置项，全局效果默认启用

### 其他

- 爱发电地址更新为 ifdian.net
- README 文档更新项目结构和配置说明

## [v1.1.0] - 2026-03-13

### 新功能

- **音乐播放器** - 新增音乐播放器组件，支持呼吸光晕效果、玻璃微珠按钮质感
- **GitHub 模块** - 新增 GitHub 项目展示和贡献图模块
- **博客集成** - 新增 RSS 博客文章集成，支持 3D 翻转卡片布局
- **主题系统** - 动态主题切换 (Auto/Light/Dark)、自定义配色方案、界面中文化
- **赞赏支持** - 新增赞赏模块，支持二维码和外部链接
- **导航增强** - 顶部导航栏、滚动进度按钮、打字机品牌效果
- **Favicon** - 支持从头像自动生成 favicon
- **分析工具** - 支持 Google Analytics、Clarity、Umami 等多种分析方案
- **公告组件** - 可配置的防诈骗警告公告
- **玻璃质感** - 玻璃光泽效果、移动端触摸手势支持

### 优化

- **性能** - 首屏性能优化、构建压缩 (JS/CSS/HTML/图片)
- **骨架屏** - 完善骨架屏覆盖范围，精确匹配布局
- **无障碍** - 规范化设计系统，改进无障碍访问
- **字体** - 升级字体策略，新增 display 衬线字体
- **配色** - 珊瑚橙配色调整、配色方案双色圆点预览

### 其他

- 支持页脚 ICP 备案号配置
- 邮箱反爬虫保护功能
- 网格动画增强
